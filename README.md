# Помощник эксперта по сертификации

Приложение для подбора стандартов на основе базы выданных сертификатов ТР ТС 004/2011 и ТР ТС 020/2011.

## Запуск

```bash
streamlit run src/app.py
Добавление новых сертификатов (локально)
Запустите приложение

В боковой панели нажмите «Загрузить» и выберите Excel-файл

Введите пароль и нажмите «Сохранить в основную базу»

Закройте приложение (Ctrl+C)

Запустите шифрование:

bash
python encrypt_data.py
Готово! Новые данные в certificates.enc.

Отправка обновлённой базы в Streamlit Cloud
После локального добавления:

bash
git add data/certificates.enc
git commit -m "Обновлена база сертификатов"
git push
Затем на Streamlit Cloud нажмите Reboot app.

Безопасность
Исходный файл certificates.xlsx не хранится в репозитории

В GitHub загружается только зашифрованный certificates.enc

Ключ расшифровки хранится в секретах Streamlit Cloud (ENCRYPTION_KEY)
